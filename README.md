* Lesson 004
	* 下記コマンドラインアプリを作成する。
		* 第一引数により処理の制御を行う。
			* "1"：足し算
			* "2"：引き算
			* "3"：掛け算
			* "4"：割り算
		* 上記の制御に沿って第二引数と第三引数を計算する。
		* それぞれの制御に応じた関数を作成する。
		* エラー処理を加える。
			* 引数が3つ未満の場合エラー。
				* "起動引数が足りません。"
			* 第一引数に"1～4"以外の値が指定された場合エラー。
				* "既定外の値が入力されました。"
			* 第二引数、第三引数に数値以外が入力された場合はエラー
				* "数値以外が入力されました。"
